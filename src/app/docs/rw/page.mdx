# Revived Witch

Welcome to the Revived Witch API documentation. This API is intended for developers who want to interact with the Revived Witch data without having to worry about parsing the files. Here you will find information on how to interact with the API.

The Revived Witch API is a RESTful API. If you want the raw data, you can find it [here](https://github.com/timeworn/revived-witch-parser)
