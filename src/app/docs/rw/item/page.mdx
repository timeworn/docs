# Item

This endpoint allows you to search for a item.

## Endpoint

- URL: `/rw/v1/item[/:id]`
- Method: `GET`
- Response: `application/json`

## Query Parameters

| Name     | Type                                                                            | Description                                                                |
| -------- | ------------------------------------------------------------------------------- | -------------------------------------------------------------------------- |
| `type`   | `string` [Weapon, Armor, Accessory, Furniture, Intimacy Item, Unique Equipment] | The type of the item.                                                      |
| `fields` | `string`                                                                        | Restricts the [fields](/docs/api-introduction#query-parameters) to return. |
| `offset` | `int`                                                                           | The amount of items you want to skip.                                      |
| `limit`  | `int`                                                                           | The amount of items to return.                                             |

### Equipment

| Name        | Type                                                                                                                                                                                                              | Description                     |
| ----------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------- |
| `score`     | `int` [540, 570, 580, 600, 610, 630, 640, 690, 720, 740, 750]                                                                                                                                                     | The score of the item.          |
| `equipSuit` | `string` [None, Frenzy, Orderly Fashion, Goddess's Blessed, Unbending Will, Complexity, Sorcerer, Inheritance of Life, Periodic Supply, Scholar, Soul-devoured Bones, Ancient Forest, Iron-willed, Peace Of Mind] | The equipment suit of the item. |

### Furniture

| Name           | Type                                                                                                                                                                                       | Description                    |
| -------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------ |
| `level`        | `int` [1-4, 6]                                                                                                                                                                             | The level of the item.         |
| `cover`        | `string` [20\*2, 20\*7, 4\*5, 4\*2, 6\*2, 2\*1, 4\*1, 2\*3, 2\*2, 1\*1, 3\*2, 3\*5, 8\*3, 4\*7, 3\*3, 6\*1, 4\*3, 1\*2, 7\*2, 5\*3, 5\*2, 4\*4, 3\*1, 3\*4, 2\*4, 8\*2, 1\*3, 7\*1, 2\*2,] | The cover of the item.         |
| `comfortPoint` | `int` [10, 15, 20, 25, 50, 90]                                                                                                                                                             | The comfort point of the item. |

### Gift

| Name       | Type                     | Description             |
| ---------- | ------------------------ | ----------------------- |
| `favour`   | `int` [100, 150, 200]    | The favour of the item. |
| `exfavour` | `int` [0, 100, 150, 200] | The favour of the item. |

### Unique Equipment

| Name    | Type                      | Description            |
| ------- | ------------------------- | ---------------------- |
| `owner` | `string` [Character Name] | The owner of the item. |
