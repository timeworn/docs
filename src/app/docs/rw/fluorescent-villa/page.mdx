# Fluorescent Villa

This endpoint allows you to search for a training.

## Endpoint

- URL: `/rw/v1/fluorescent-villa[/:id]`
- Method: `GET`
- Response: `application/json`

## Query Parameters

| Name      | Type                                                                                                                                                                            | Description                                                                |
| --------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------- |
| `time`    | `int` [120, 240, 480, 600, 720, 960]                                                                                                                                            | The time of the training.                                                  |
| `rarity`  | `int` [1-4]                                                                                                                                                                     | The rarity of the training.                                                |
| `expGain` | `int` [6000, 12000, 24000, 8000, 16000, 32000, 18000, 36000, 72000, 90000, 48000, 96000, 144000, 180000, 192000, 288000, 384000, 50400, 100800, 201600, 302400, 403200, 151200] | The experience gained from the training.                                   |
| `fields`  | `string`                                                                                                                                                                        | Restricts the [fields](/docs/api-introduction#query-parameters) to return. |
| `offset`  | `int`                                                                                                                                                                           | The amount of trainings you want to skip.                                  |
| `limit`   | `int`                                                                                                                                                                           | The amount of trainings to return.                                         |
