# Avatar

This endpoint allows you to search for a in-game avatar.

## Endpoint

- URL: `/rw/v1/account/avatar[/:id]`
- Method: `GET`
- Response: `application/json`

## Query Parameters

| Name     | Type     | Description                                                                |
| -------- | -------- | -------------------------------------------------------------------------- |
| `fields` | `string` | Restricts the [fields](/docs/api-introduction#query-parameters) to return. |
| `offset` | `int`    | The amount of avatars you want to skip.                                    |
| `limit`  | `int`    | The amount of avatars to return.                                           |
