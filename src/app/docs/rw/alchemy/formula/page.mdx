# Formula

This endpoint allows you to search for a alchemy formula.

## Endpoint

- URL: `/rw/v1/alchemy/formula[/:id]`
- Method: `GET`
- Response: `application/json`

## Query Parameters

| Name          | Type               | Description                                                                |
| ------------- | ------------------ | -------------------------------------------------------------------------- |
| `type`        | `int` [1-2]        | The type of the alchemy formulas.                                          |
| `unlockLv`    | `int` [1-3]        | The unlock level of the alchemy formulas.                                  |
| `emotionCost` | `int` [2-4]        | The emotion cost of the alchemy formulas.                                  |
| `getExp`      | `int` [10, 15, 20] | The get exp of the alchemy formulas.                                       |
| `fields`      | `string`           | Restricts the [fields](/docs/api-introduction#query-parameters) to return. |
| `offset`      | `int`              | The amount of alchemy formulas you want to skip.                           |
| `limit`       | `int`              | The amount of alchemy formulas to return.                                  |
