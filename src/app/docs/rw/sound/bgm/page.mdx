# Background Music

This endpoint allows you to search for a background music.

## Endpoint

- URL: `/rw/v1/sound/bgm[/:id]`
- Method: `GET`
- Response: `application/json`

## Query Parameters

| Name     | Type     | Description                                                                |
| -------- | -------- | -------------------------------------------------------------------------- |
| `fields` | `string` | Restricts the [fields](/docs/api-introduction#query-parameters) to return. |
| `offset` | `int`    | The amount of background music you want to skip.                           |
| `limit`  | `int`    | The amount of background music to return.                                  |
