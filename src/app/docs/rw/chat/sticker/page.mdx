# Sticker

This endpoint allows you to search for stickers.

## Endpoint

- URL: `/rw/v1/chat/sticker[/:id]`
- Method: `GET`
- Response: `application/json`

## Query Parameters

| Name     | Type     | Description                                                                |
| -------- | -------- | -------------------------------------------------------------------------- |
| `fields` | `string` | Restricts the [fields](/docs/api-introduction#query-parameters) to return. |
| `offset` | `int`    | The amount of stickers you want to skip.                                   |
| `limit`  | `int`    | The amount of stickers to return.                                          |
