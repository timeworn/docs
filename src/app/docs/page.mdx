# Introduction

Timeworn offers a complete range of free, public services. Here you will find documentation, guides, and references on how interact with our services to create your applications.

## Prerequisites

- Basic knowledge of HTTP.
- Basic knowledge of JSON responses.
- Basic knowledge of JavaScript or any other programming language that can make HTTP requests.

If you are not familiar with these concepts, we recommend you take some time to learn them. Here are some resources to help you get started:

- [MDN HTTP Basics](https://developer.mozilla.org/en-US/docs/Web/HTTP) - MDN's HTTP guide and documentation
- [MDN JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript) - MDN's JavaScript guide and documentation.
- [Stack Overflow](https://stackoverflow.com/) - A question and answer site for programmers.
- [ChatGPT](https://chat.openai.com/) - An AI language model that can help you with programming.
- [Google](https://www.google.com/) - Your best friend.

All services are free to use, but we do have some guidelines. Please read our [Terms of Use](/docs/terms-of-use) before using our services.
